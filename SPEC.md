## 3. Design System

### Colors
| Role | Color | Hex |
|------|-------|-----|
| Background | Deep Dark | #0B0F19 |
| Cards | Dark Gray | #111827 |
| Primary | Indigo | #6366F1 |
| Secondary | Cyan | #22D3EE |
| CTA | Amber | #F59E0B |
| Success | Green | #10B981 |
| WhatsApp | Green | #25D366 |
| Error | Red | #EF4444 |
| Text Primary | White | #FFFFFF |
| Text Secondary | Gray | #9CA3AF |

### Typography
- **Headings:** Inter (Google Font), weights 600-900
- **Body:** Inter, weights 300-500
- **Accent/Code:** JetBrains Mono

### Effects
- Glow borders on cards (primary color)
- Gradient overlays on featured cards
- Glassmorphism (backdrop blur)
- Floating animations on profile images
- Hover scale + glow effects

---

## 4. Animations

### Scroll Animations (Intersection Observer)
- Fade-in-up for all cards and sections
- Staggered entrance delays for grid items
- Skill bars fill animation on scroll

### Continuous Animations
- Hero gradient orbs floating
- Particle background
- Profile image floating
- Pulse animation on availability badge

### Hover Effects
- Cards scale up with enhanced glow
- Buttons translate up with glow intensify
- Navigation links underline animation

---

## 5. Navigation

- Sticky navbar with blur background on scroll
- Logo with monospace formatting
- Nav links: Problems, Services, Tutoring, Projects, Skills, Let's Talk
- Mobile hamburger menu
- Scroll progress bar at top

---

## 6. Responsive Breakpoints

- **Desktop:** 1200px+ (full layout)
- **Tablet:** 968px (stacked hero, 2-column grids)
- **Mobile:** 768px (single column, hamburger menu)
- **Small Mobile:** 480px (compact spacing)
